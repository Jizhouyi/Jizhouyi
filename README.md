## 🚀 主要项目

### 1. **多阶段奖励模块系统** | Multi-stage-awarding-module
一个企业级的微服务架构项目，实现了复杂的用户积分和奖励管理系统。该项目采用模块化设计，包含积分管理、虚拟货币、音乐播放记录、红包系统等多个独立模块。

**技术栈**：Java 17 · Spring Boot 3.4 · MyBatis · MySQL

**核心特性**：
- 模块化架构设计，支持独立部署和扩展
- 完整的积分生命周期管理（创建、发放、消费、撤销）
- 虚拟货币系统支持多种交易场景
- 红包系统实现公平的资源分配算法
- 详细的操作日志和审计追踪

[查看项目 →](https://github.com/Jizhouyi/Multi-stage-awarding-module)

### 2. **草莓病害图像分割系统** | Strawberry-Disease-Segmentation
一个基于深度学习的农业应用项目，专注于草莓病害的自动识别和分割。该系统能够精确定位和识别草莓上的各种病害，为农业生产提供数据支持。

**技术栈**：Python · Deep Learning · Image Segmentation · Computer Vision

**核心特性**：
- 高精度的病害识别和分割算法
- 支持多种草莓病害类型的检测
- 实时推理能力，适合田间应用
- 可视化结果展示和数据分析

[查看项目 →](https://github.com/Jizhouyi/Strawberry-Disease-Segmentation)
